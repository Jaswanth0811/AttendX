## 📱 About

AttendX is a high-end attendance tracking solution designed for students who value aesthetics and functionality. It features a modern dashboard, automated timetable-based tracking, and a smart analytics engine that predicts your future attendance status based on custom goals.

The app is built natively for Android using **Kotlin** and **Jetpack Compose**, following Clean Architecture principles to ensure a smooth and reliable user experience.

---

## ✨ Features

- **🚀 Smart Import (v4.6.0)**: 
  - **AI OCR Scanning**: Automatically extract your schedule from photos using Google ML Kit.
  - **Excel & PDF Support**: Import subjects and timings directly from digital documents (.xls, .xlsx, .pdf).
  - **CSV Integration**: Support for standard comma-separated files.
- **📈 Smart Predictions**: Real-time calculations of how many classes you can miss or need to attend to reach your goal.
- **📅 Dynamic Timetable**: Automated attendance prompts based on your daily schedule and easy configuration.
- **🗓️ Calendar View**: Deep historical tracking with the ability to mark holidays and adjust records retroactively.
- **📊 Rich Analytics**: Visual insights with progress rings, bar charts, and subject-wise details.
- **🎨 Modern UI**: A premium experience with glassmorphism, smooth animations, and Material 3 design.
- **💾 Data Management**: Secure Backup/Restore functionality and flexible data exports.

---

## 🛠️ Tech Stack & Tools

### Languages
- **Kotlin**: The primary language for modern Android development.
- **SQL (Room)**: Used for high-performance local data persistence.

### Frameworks & Libraries
- **Jetpack Compose**: For building a reactive and beautiful UI.
- **Material 3**: Google's latest design system for consistent aesthetics.
- **Dagger Hilt**: For robust dependency injection.
- **Google ML Kit**: Powering the AI-based Text Recognition (OCR).
- **CameraX**: For seamless in-app camera interactions.
- **Apache POI**: To handle Excel file parsing (.xls, .xlsx).
- **PdfBox-Android**: For extracting data from PDF documents.
- **Coroutines & Flow**: For efficient asynchronous programming.

### Tools Used
- **Android Studio (Ladybug/Latest)**: The primary IDE.
- **Gradle (Kotlin DSL)**: For project build and dependency management.
- **KSP (Kotlin Symbol Processing)**: For faster build times with Room and Hilt.
- **Git & GitHub**: For version control and releases.

## 👨‍💻 Development

```bash
# Clone the repository
git clone https://github.com/Jaswanth0811/AttendX.git

# Navigate to project
cd AttendX

# Build the APK
./gradlew assembleDebug
```
